**1.1 THE MAGIC SPELL**

**மந்திரச் சொல்**

**English**

A computer is essentially a box of metal and electricity. On its own, it
doesn\'t know how to do anything---it can\'t play videos, it can\'t
calculate math, and it certainly can\'t understand your voice. It needs
instructions.

Think of a computer program like a **Cooking Recipe**. If you want to
make Biryani, you follow a set of steps: *Cut onions, fry spices, add
rice.* If you miss a step, the taste changes. Similarly, **Python** is
the language we use to write \"recipes\" for the computer. We call these
recipes **Programs**.

A **Programming Language** is simply a set of rules that both humans and
computers can understand. Just as English has grammar rules (subject,
verb, object), Python has syntax rules. If you follow the rules
correctly, the computer understands your command.

Why is it called Python? The creator, Guido van Rossum, named it after
the British comedy show *Monty Python\'s Flying Circus*---not after the
snake! But we like the snake metaphor because the symbol `>>>` in Python
looks like a snake head waiting for your next command.

**Tamil**

கணினி என்பது அடிப்படையில் உலோகம் மற்றும் மின்சாரம் நிறைந்த ஒரு பெட்டி. தானாகவே

(Instructions)** தேவை.

ஒரு கம்ப்யூட்டர் ப்ரோக்ராமை **சமையல் குறிப்பு (Recipe)** போல நினைத்துக் கொள்ளுங்கள்.
நீங்கள் பிரியாணி செய்ய விரும்பினால், சில வழிமுறைகளைப் பின்பற்றுவீர்கள்: *வெங்காயத்தை
நறுக்கு, மசாலா வதக்கு, அரிசி சேர்.* ஒரு படியைத் தவறவிட்டால், சுவை மாறிவிடும்.

அதேபோல, **பைதான் (Python)** என்பது கணினிக்கு \"சமையல் குறிப்புகளை\" எழுதப்
பயன்படுத்தும் ஒரு மொழி. இந்த குறிப்புகளைத்தான் நாம் **நிரல்கள் (Programs)** என்று
அழைக்கிறோம்.

**நிரல் மொழி (Programming Language)** என்பது மனிதர்களுக்கும் கணினிக்கும்
புரிந்துகொள்ள முடிக்கூடிய விதிகளின் தொகுப்பாகும். தமிழில் வாக்கியங்களுக்கு விதிகள்
இருப்பது போல, பைதானுக்கும் **வாக்கியமைப்பு விதிகள் (Syntax rules)** உண்டு. இந்த
விதிகளைத் சரியாகப் பின்பற்றினால், கணினி உங்கள் கட்டளையைப் புரிந்து கொள்ளும்.

இதை ஏன் \"Python\" என்று அழைக்கிறார்கள்? பைதானை உருவாக்கியவர், குய்டோ வேன் ரோசம்

**1.4 YOUR FIRST COMMAND**

**உங்கள் முதல் கட்டளை**

**English**

In traditional books, the first program is always \"Hello World\". But
we are **innovators!** Let\'s make Python speak with a Tamil flavor and
some personality. We will use a **Function** called `print()`.

Type the following exactly as shown next to the `>>>` prompt and hit
Enter:

> print(\"Vanakkam Ulagam!\")\
> Vanakkam Ulagam!

**Congratulations!** You just wrote your first Python command. The
computer understood your instruction and displayed the text on the
screen.

**What just happened?**

- `print` - This tells Python to display (or \"print\") something on the
  screen.

- `(...)` - The parentheses are like a container. They hold what you
  want to print.

- `"..."` - The quotation marks tell Python that this is **text**, not a
  number or command.

**Breaking it Down:**

1.  You typed the command `print("Vanakkam Ulagam!")`

2.  Python read your command.

3.  Python executed (ran) your command.

4.  Python displayed the result: `Vanakkam Ulagam!`

This is the **fundamental cycle** of programming:

**Type → Python Reads → Python Executes → Python Displays Result**

**2.1 WHAT IS A VARIABLE?**

**மாறி (Variable) என்றால் என்ன?**

**English**

Imagine you are moving to a new house. You have many items: books,
kitchenware, and clothes. You don\'t just throw them all in a pile; you
put them in **boxes**. To remember what is inside, you write a label on
the box: \"Kitchen\", \"Books\", or \"Shoes\".

In Python, a **Variable** is exactly like that box. It is a container
where we store data (information) so we can use it later.

- The **Box** = The Variable

- The **Label** = The Variable Name

- The **Item inside** = The Value

**Example:**

> score = 100\
> name = \"Raman\"

Here, we created a box labeled `score` and put the number `100` inside.
We created another box labeled `name` and put the text `"Raman"` inside.

**2.2 RULES FOR NAMING YOUR BOXES**

**பெட்டிகளுக்குப் பெயரிடுவதற்கான விதிகள்**

**English**

You can\'t just give any name to your variable. Python has strict rules,
just like passport officials!

**The 3 Golden Rules:**

1.  **Start with a letter or underscore (\_):** You can use `age` or
    `_secret`, but NOT `1st_place`.

2.  **No Spaces:** You cannot name a variable `my name`. Instead, use an
    underscore: `my_name`.

3.  **Case Sensitive:** `Age`, `age`, and `AGE` are three *different*
    boxes to Python.

**Valid Names (Good):** `hero_health`, `player2`, `total_money`\
**Invalid Names (Bad):** `2player` (starts with number), `hero health`
(has space), `class` (reserved word)

**3.2 IF, ELSE, AND ELIF**

**If, Else, மற்றும் Elif**

To cast this spell correctly, you must follow the strict rules of Python
syntax. If you make a mistake, the spell will fail (and you will get a
Syntax Error!). Here is the formula:

python

if condition:

*\# This code runs ONLY if the condition is True*

statement_1

statement_2

**The 3 Golden Rules:**

1.  The Condition: This is the question you are asking (e.g., age \>
    18). It must be a question that can be answered with a simple
    \"Yes\" (True) or \"No\" (False).

2.  The Colon (:): CRITICAL! The line must end with a colon. This tells
    Python, \"Listen, I am about to tell you what to do next.\" It is
    like the handle on the door.

3.  Indentation (The Space): Notice the gap before the statements? In
    Python, you MUST press the Tab key (or type 4 spaces) to push the
    code to the right. This indentation tells Python that these specific
    lines belong to the if statement. If you don\'t indent, Python
    thinks they are just regular code outside the gate.

** பொன்விதிகள்**\]

1.  நிபந்தனை (Condition): இதுதான் நாம் கேட்கும் கேள்வி (எ.கா: age \> 18). இதன்
    விடை True அல்லது False ஆக மட்டுமே இருக்க வேண்டும்.

2.  முக்காலி (:): வரியின் இறுதியில் கண்டிப்பாக ஒரு முக்காலி (Colon) வைக்க
    வேண்டும். இதுதான் \"கதவைத் திறக்கும் சாவி\". இதை மறப்பது பைத்தானில் செய்யப்படும்
    பொதுவான தவறு!

3.  உள்ளிடக்கம் (Indentation): அடுத்த வரியை ஆரம்பிக்கும் முன் Tab விசையை அழுத்தி
    சற்று தள்ளி எழுத வேண்டும். இந்த இடைவெளிதான் \"இந்தக் வரிகள் if-க்குச்
